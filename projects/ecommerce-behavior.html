SQL BigQuery

E-commerce User Behavior & Conversion Insights

SQL · BigQuery · Google Analytics · Funnel Analysis

An advanced SQL project using Google BigQuery to analyze Google Analytics e-commerce sessions. Using CTEs, window functions, and data segmentation, the project uncovers conversion bottlenecks, contrasts Purchaser vs Non-Purchaser behavior, and identifies the highest-value traffic sources and product categories.

📅 2024 🛠️ SQL, BigQuery 🔍 Google Analytics Sessions

01. Business Challenges

An e-commerce business was generating significant website traffic but struggling to understand why most visitors weren't converting. Google Analytics provides raw session data, but without structured SQL analysis, the team couldn't answer critical questions about user behavior and funnel performance.

🔍 Conversion Mystery

The overall conversion rate was low, but no one could pinpoint where in the funnel users were dropping off or which segments were underperforming.

👥 User Segmentation Gap

No clear data comparing how Purchasers and Non-Purchasers behaved differently — page views, session duration, bounce rate, and engagement patterns.

📊 Traffic Source ROI

Marketing spent budget across multiple channels but lacked data on which traffic sources delivered the highest-value users and best conversion rates.

02. My Approach

I used advanced SQL techniques in Google BigQuery to transform raw Google Analytics session data into actionable insights — working directly with the bigquery-public-data.google_analytics_sample dataset.

1

Data Exploration

Explored the nested GA schema structure using UNNEST() to flatten hits, products, and custom dimensions into queryable rows. Assessed data quality and coverage across time periods.

2

Behavioral Segmentation

Used CTEs and window functions to classify users into Purchaser vs Non-Purchaser segments, then compared page views, bounce rates, session duration, and revenue per session.

3

Funnel & Cohort Analysis

Built conversion funnel queries to identify drop-off points, and cohort analyses to track how user behavior changes over time and across traffic sources.

03. SQL Analysis Deep Dive

The analysis was structured into multiple query modules, each answering a specific business question:

📊 Traffic Analysis

  • Monthly active users and session counts
  • Traffic source breakdown (organic, paid, direct, referral)
  • Bounce rate by source and landing page

🛒 Conversion Funnel

  • Product view → Add to cart → Purchase rates
  • Drop-off analysis at each funnel stage
  • Revenue per session by user segment

Key SQL Techniques Used

  • UNNEST() — Flattening nested GA hit-level and product-level data
  • WITH (CTEs) — Building modular, readable query chains for multi-step analysis
  • NTILE() and window functions — User scoring and percentile-based segmentation
  • CASE WHEN — Purchaser vs Non-Purchaser behavioral classification
  • Cohort analysis — Tracking user retention and behavior changes over time

04. Key Insights

The SQL analysis revealed clear, actionable patterns in user behavior and conversion:

🛒 Funnel Drop-Off

The biggest conversion gap is between Product View → Add to Cart — a large percentage of users browse but never add items, suggesting product page optimization opportunities.

👥 Purchaser Behavior

Purchasers view significantly more pages per session and spend longer on-site compared to Non-Purchasers — engagement depth is a strong conversion predictor.

🔗 Traffic Source Quality

Organic search delivers the highest-value users (highest revenue per session), while some paid channels show high traffic volume but poor conversion rates.

📈 Seasonal Patterns

Monthly cohort analysis reveals clear seasonal peaks in conversion rates — suggesting optimal timing for marketing campaigns and promotions.

05. Business Impact

The analysis provides a data-driven roadmap for improving e-commerce conversion rates:

🎯 Product Page Optimization

Focus UX improvements on the product page → cart transition — the single biggest conversion bottleneck identified in the funnel analysis.

💰 Channel Reallocation

Shift marketing budget toward high-conversion traffic sources (organic) and away from high-volume but low-converting paid channels.

📊 Engagement-Based Targeting

Use page-view depth and session duration as signals to identify high-intent users for retargeting campaigns.
